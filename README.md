Description
Hamker
- 체크리스트(Todo-List), 가계부, 일정관리 등 개인 다이어리를 한 눈에 확인, 관리가 가능하며
사용자가 직접 커스텀이 가능한 다이어리 웹앱개발을 목표!

Getting Started
- index.html 실행
* 크롬(chrome)에 적합
* library를 사용하지 않고 제작하였습니다 (2023. 01. 03 기준 - 변경 예정)

Important Files
- manifest.json
- serviceWorker.js
* 위 두 파일이 없으면 PWA설치가 불가능합니다.

Used Resources
- JQuery
- IndexedDB
